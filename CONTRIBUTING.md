# Guias de Contribuição 

Primeiramente, obrigado por se interessar em contribuir para o **Pedido de Namoro Irrecusável**! É o apoio da comunidade que torna este projeto mais criativo e divertido.

Para manter a qualidade e a organização do projeto, pedimos que siga estas diretrizes.

---

## Como posso contribuir?

### 1. Relatando Bugs 
Se você encontrar algum erro técnico ou algo que não funcione como esperado:
- Verifique se o bug já foi relatado nas [Issues](https://github.com/StellaKarolinaNunes/Website-pedido-de-namoro-irrecusavel/issues).
- Caso não, abra uma nova issue descrevendo o problema, passos para reproduzir e, se possível, prints de tela.

### 2. Sugerindo Novas Funcionalidades 
Tem uma ideia para tornar o pedido ainda mais "irrecusável"?
- Abra uma issue do tipo "Feature Request".
- Explique como a nova funcionalidade ajudaria o projeto e o casal.

### 3. Enviando Alterações (Pull Requests) 
Se você quer colocar a mão no código:
1. Faça um **Fork** do repositório.
2. Clone o seu fork: `git clone https://github.com/SEU_USUARIO/Website-pedido-de-namoro-irrecusavel.git`.
3. Crie uma branch para sua modificação: `git checkout -b feature/minha-melhoria`.
4. Faça suas alterações seguindo os padrões de código do projeto.
5. Faça o commit de suas mudanças: `git commit -m 'feat: adiciona efeito X'`.
6. Envie para o GitHub: `git push origin feature/minha-melhoria`.
7. Abra um **Pull Request** para a branch `main` do repositório original.

---

##  Padrões de Projeto

- **HTML:** Mantenha a semântica e identação limpa.
- **CSS:** Utilize variáveis (se possível) e mantenha o estilo premium/glassmorphism.
- **JavaScript:** Código bem comentado e sem variáveis globais desnecessárias.
- **Imagens:** Devem ser otimizadas para web para não sobrecarregar o carregamento.

---

## Código de Conduta

Ao participar deste projeto, você concorda em manter um ambiente respeitoso, acolhedor e livre de qualquer tipo de assédio. Lembre-se: este é um projeto focado em **amor e diversão**!
